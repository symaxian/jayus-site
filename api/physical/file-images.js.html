Function Details

function get

Image get(String filepath)
Returns the specified image.
The image will still be returned if it is not yet loaded.
Parameters:
filepath

function has

Image has(String filepath)
Returns whether the specified image has been requested.
Note that this does not mean that the image is loaded, only that the image object is available and that is has been requested.
Parameters:
filepath

function isLoaded

Boolean isLoaded(String filepath)
Returns whether the specified image is loaded or not.
Parameters:
filepath

function load

load(String filepath, Function handler)
Loads the specified image.
An image will not be loaded twice.
The optional callback handler will be executed when the image is loaded, or immediately if it is already loaded.
Parameters:
filepath
handler Optional

function loadAll

loadAll(Array filepaths, Function handler)
Loads the specified images.
The optional callback handler will be executed when all the images have been loaded, or immediately if they have all already been loaded.
Parameters:
filepaths
handler Optional
