function load

load(String filepath, Function handler)
Loads and parses the specified JSON file.
The optional callback handler will be executed when the file is loaded, or immediately if it is already loaded.
The arguments sent to the handler are the filepath followed by the retrieved object and the XMLHttpRequest object.
An object will not be loaded twice.
Parameters:
filepath
handler Optional

function loadAll

loadAll(Array filepaths, Function handler)
Loads the specified objects.
The optional callback handler will be executed when all the objects have been loaded, or immediately if they have all already been loaded.
Parameters:
filepaths
handler Optional
