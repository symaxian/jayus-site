Property Details

property Object TYPES

An enumeration of the types that the jayus.debug.is() method can check against.
Only found in the "debug" versions of jayus.
Values are: DEFINED, BOOLEAN, STRING, FUNCTION, NUMBER, OBJECT, ARRAY, CONTEXT, COLOR, BRUSH, MATRIX, LIST, ENTITY, POINT, CIRCLE, POLYGON, PATH, SURFACE, SCENE, SHAPE, RECTANGLE, BUFFER, ANIMATOR, CANVAS.

property Array animators

An array of running animators.
Only animators in this array will be ticked(updated) by jayus.
Each animator adds itself to this array when started and is removed when finished.
Do not modify.

property Entity cursorFocus

The Entity that has cursor focus.

property Boolean declareChildrenAsProperties

Determines whether children will be mapped onto their parent object as properties, using their ID as the key.
The child must have an ID and it may not collide with any pre-existing property, else an error will be displayed if using the 'debug' version of jayus.
One way ensure there are never any collisions with built-in properties is to capitalize all of your entity ID's.
Default is true.

property Array displays

An array of the jayus displays on the page.

property Display focusedDisplay

The focused display, or null.

property Object fontCache

Font cache storage.
Holds the character widths for each cached font.
Do not modify.

property Number fontCacheMaxChar

The upper bound for the codepoints of characters that will be cached when caching a font.
The characters cached are those with a codepoint between fontCacheMinChar and this property.
Default is 255.

property Number framerate

The desired framerate.
This framerate is only used if the browsers requestAnimationFrame() function is unavailable.
After changing this property jayus must be restarted if it is running.
Default is 60

property Boolean hasFocus

Whether or not a jayus Display is the focused page element.

property Array identifiedObjects

An array of identified objects.

property Boolean keepTrackOfObjects

Determines whether identified entities will be stored in jayus.identifiedObjects or not.
Default is false.

property Number lastFrameTime

The time(in epoch time) when the previous frame was drawn, used in calculating frames per second.

property Number lastStepTime

The time(in epoch time) when jayus was previously stepped(updated).

property Boolean running

Whether jayus is runnning or not.
Do not modify, use jayus.start() and jayus.stop().

property Number steprate

The desired animation steps per second.
After changing this property jayus must be restarted if it is running.
Default is 60

property Boolean useReqAnimFrame

Whether to use the requestAnimationFrame function if available.
If set to false or the requestAnimationFrame function is not available then then jayus will attempt to maintain the framerate found in jayus.framerate.

function detectBrowser

Object detectBrowser()
Returns browser and OS info.
The returned object's properties(strings) are 'browser', 'version', and 'os'.
The value "unknown" is used if unable to detect a property.

Self init()
Initializes Jayus.
