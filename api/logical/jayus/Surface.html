Class jayus.Surface

class jayus.Surface
An Entity that represents an editable image.
The Surface class acts as a wrapper for a hidden canvas element.

function blurImage

Self blurImage(Number intensity)
Blurs the image on the surface.
Performs a gaussian blur with the specified intensity.
Blurs the alpha value as well.
Parameters:
intensity

function clear

Self clear()
Clears the surface.
Fills the surface with transparent black.

function exportPNG

Self exportPNG()
Export the surface to a png image and open it in a new tab, browser permitting.
Note that exporting a virtual surface takes longer, as the virtual region must be copied then exported.
