function getAllChildren

Array getAllChildren(Boolean parentsAfter)
Returns every child under the group in an array in the order they appear.
Essentially "flattens" the scene-graph under the group.
This group is not included in the array.
If parentsAfter is true then parents will appear after their children in the array, otherwise they come first.
Parameters:
parentsAfter Optional
