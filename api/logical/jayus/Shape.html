Class jayus.Shape

Direct Known Subclasses:
jayus.Circle, jayus.Path, jayus.Polygon
class jayus.Shape
An abstract class that represents a geometric shape.
Note that a Shape is not an Entity, it must be placed within a PaintedShape entity to be drawn.
