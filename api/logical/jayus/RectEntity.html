property Number height

The height of the Entity.

property Boolean negateBlur

Enables manually rendering the buffer of the entity to negate the blurring effect when scaled up.
If enabled, the buffer for the entity will be drawn pixel-by-pixel.
This serves to negate the blurring effect that occurs when drawing a scaled-up entity.
This alternate rendering method is MUCH slower as each pixel is drawn individually.
Default is false.

property Boolean roundX

Whether to attempt to keep the Entity horizontally aligned to the pixel-grid.
This will keep images/shapes from being rendered at fractions of a pixel.
This property applies to the translation of the entity(and thus its children).
This may not force alignment on the background or any child entities.
Default is false.

property Boolean roundY

Whether to attempt to keep the Entity vertically aligned to the pixel-grid.
This will keep images/shapes from being rendered at fractions of a pixel.
This property applies to the translation of the entity(and thus its children).
This may not force alignment on the background or any child entities.
Default is false.

property Number width

The width of the Entity.
Do not modify.

property Number x

The x position of the Entity.
Default is 0.
Do not modify.

property Number y

The y position of the Entity.
Default is 0.
Do not modify.
