Class jayus.Frame

Extends jayus.RectEntity , jayus.Wrapper
Direct Known Subclasses:
jayus.EditableFrame, jayus.FlexibleFrame
class jayus.Frame
An entity that serves as a padded border around an entity.
Whenever the frame is resized, it resizes its child to fit.
Requires a resizable entity for a child.
