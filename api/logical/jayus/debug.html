Namespace jayus.debug

namespace jayus.debug
Holds properties and methods dedicated to debugging routines.
Only found in the "debug" versions of jayus.

Property Summary
Boolean pauseOnEscape
Whether or not to pause jayus when the Escape button is pressed in debug mode.

Function Summary

Returns whether or not the sent variable is of the specified type.

Returns a string representation of the given variable for debugging purposes.

Property Details

property Boolean pauseOnEscape

Whether or not to pause jayus when the Escape button is pressed in debug mode.
Default is true
This property and its functionality only exists in debug mode.

Function Details

function is

Boolean is(Number type, * v)
Returns whether or not the sent variable is of the specified type.
Types are specified using the jayus.TYPES enumerations.
Types include: Defined, Boolean, String, Function, Number, Object, Array, Context, Color, Brush, Matrix, List, Entity, Point, Circle, Polygon, Path, Surface, Scene, Shape, Rectangle, Buffer, Animator, Canvas.
