Class jayus.Display

Extends jayus.Scene
class jayus.Display
A Scene that can be placed onto the page.
The display class also manages the cursor events for itself and it's children.
Each display is refreshed by jayus each frame if needed.

property Boolean pendMousemoveEvents

Whether or not to delay the processing of mousemove events until actually required.
Unless your application requires extreme cursor movement tracking accuracy, it is highly recommended to leave this option enabled. When enabled, the processing of the native mousemove event is postponed until either the start of the next frame, or when another cursor event is fired(such as a button press). This can result in the a reduction of time and garbage(discarded objects) spent processing and responding to redundant cursor movement events.
Default is true.

property Object pendingMousemoveEvent

The native event object for the pending mousemove event.
Do not modify.

property Boolean suppressContextMenu

Whether or not to keep the context menu from appearing when the canvas is right clicked on.
Default is true.

property Boolean suppressSelection

Whether or not to suppress the user from selecting the canvas element.
Uses the -webkit-user-select and -moz-user-select CSS attributes.
Default is true.
