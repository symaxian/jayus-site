About

Jayus

Jayus started a while ago, out of the need to build a feature-rich API on top of the very simplistic 2d canvas drawing interface. What started out as a few global objects housing some rudimentary routines eventually grew over time into a full-fledged scenegraph. It is not time-tested, nor close to being finished, or as elegant or orthogonal as I would like. But I feel that with a strong base and some flexibility, it may grow into something very deserving of recognition.

The Tools

JSFiddle

JSFiddle is a great in-browser sandbox for web pages, allowing HTML, CSS, JavaScript and more to be edited and updated instantly right in a browser. Due to it's flexibility and ease of use I've used it extensively to test and showcase Jayus.

JSFiddle »

jGrouseDoc

The API documentation is automatically generated from block comments in the source code, courtesy of jGrouseDoc. Though seemingly forgotten, it is an excellent documentation generator that I highly recommend.

jGrouseDoc »

Closure Compiler

Despite it's very misleading name(come on Google, really?), the closure compiler does not compile closures. It minimizes JavaScript source code, compressing and trimming multiple source files into one. As Jayus is a large project consisting of many source files, the Closure Compiler is used to minimize these to reduce the filesize and increase performance. For more info see the compilation article.
