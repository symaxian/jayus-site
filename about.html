About

Jayus

Jayus started a while ago, out of the need to build a feature-rich API on top of the very simplistic 2d canvas drawing interface. What started out as a few global objects housing some rudimentary routines eventually grew over time into a full-fledged scenegraph. It is not time-tested, nor close to being finished, or as elegant or orthogonal as I would like, but for the most part it is well designed and built on quite a few powerful concepts, and may grow into something very deserving of recognition.

The Tools

jGrouseDoc

The API documentation is automatically generated from block comments in the source code, courtesy of jGrouseDoc. Though seemingly forgotten, it is an excellent documentation generator that I highly recommend.

jGrouseDoc »

Closure Compiler

Despite it's very misleading name(come on Google, really?), the closure compiler does not compile closures(which is a name for another programming concept). It minimizes JavaScript source code, compressing and trimming multiple source files into one. As Jayus is a large project consisting of many source files, the Closure Compiler is used to minimize these to reduce the filesize and increase performance. For more info see the compilation article.
