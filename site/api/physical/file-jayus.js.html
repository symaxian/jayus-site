Property Details

property Object SHAPES

An enumeration of the types that the jayus.intersectTest() method can check against.
Values are: LIST, CUSTOM, POINT, CIRCLE, RECTANGLE, POLYGON, PATH.

property Object TYPES

An enumeration of the types that the jayus.debug.is() method can check against.
Only found in the "debug" versions of jayus.
Values are: DEFINED, BOOLEAN, STRING, FUNCTION, NUMBER, OBJECT, ARRAY, CONTEXT, COLOR, BRUSH, MATRIX, LIST, ENTITY, POINT, CIRCLE, POLYGON, PATH, SURFACE, SCENE, SHAPE, RECTANGLE, BUFFER, ANIMATOR, CANVAS.

property Array animators

An array of running animators.
Only animators in this array will be ticked(updated) by jayus.
Each animator adds itself to this array when started and is removed when finished.
Do not modify.

property Boolean enabled

Whether Box2D support is enabled.
Do not modify.

property Array physicalEntities

An array of entities that have a Box2D body.
Do not modify.

property Array positionIterations

The number of position iterations to step Box2D to.
Default is 10.
Do not modify.

property Array velocityIterations

The number of velocity iterations to step Box2D to.
Default is 10.
Do not modify.

property Array worlds

An array of Box2D worlds.
Do not modify.

property Array blueComponents

The blue component values for the colors.

property Array cssNames

The functional names of the colors, recognized by browsers.
Ordered alphabetically.
Not available until jayus is initialized.

property Array displayNames

The display names of the colors.
Ordered alphabetically.

property Array greenComponents

The green component values for the colors.

property Array redComponents

The red component values for the colors.

property Entity cursorFocus

The Entity that has cursor focus.

property Boolean pauseOnEscape

Whether or not to pause jayus when the Escape button is pressed in debug mode.
Default is true
This property and its functionality only exists in debug mode.

property Array displays

An array of the jayus displays on the page.

property Display focusedDisplay

The focused display, or null.

property Object fontCache

Font cache storage.
Holds the character widths for each cached font.
Do not modify.

property Number fontCacheMaxChar

The upper bound for the codepoints of characters that will be cached when caching a font.
The characters cached are those with a codepoint between fontCacheMinChar and this property.
Default is 255.

property Boolean frameIntervalRunning

Whether an upcoming call to jayus.step() is expected.
Do not modify.

property Number framerate

The desired framerate.
Default is 60

property Boolean hasFocus

Whether or not a jayus Display is the focused page element.

property Number interval

The id for the main frame loop interval.

property Number lastFrameTime

The time(in epoch time) when the previous frame began, used in calculating frames per second.

property Boolean running

Whether jayus is runnning or not.
Do not modify, use jayus.start() and jayus.stop().

property Boolean useReqAnimFrame

Whether to use the requestAnimationFrame function if available.
If set to false or the requestAnimationFrame function is not available then then jayus will attempt to maintain the framerate found in jayus.framerate.

Function createClass(Object props)
Creates a constructor function, which in JavaScript is essentially a class.
The returned constructor function calls the init method with any arguments passed to it.
Any number of superclasses may be sent, the new constructor function will have inherited from all of them, in order.

function init

Self init()
Initializes Jayus.
Note that this method does not start jayus, the display canvas will appear on the page but will be blank until jayus is started.
Jayus does not need to be running to accept input, input events should be tracked and fired from this point on.

function step

step()
Calls the main client function, updates any running animations, and refreshes the displays.
Procedure:
- Fire the frame event
- Update the fps
- Process any pending events
- Tick the running animators
- Step Box2D, if enabled
- Refresh the displays

function stop

stop()
Stops jayus.
Input events are still received and handled.
