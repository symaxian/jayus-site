function arc

Self arc(Number x, Number y, Number radius, Number startAngle, Number endAngle, Boolean anticlockwise)
Adds an arc to the path.
The arc is defined by its center, radius, and start/end angle.
A line is drawn from the current point to the starting point of the arc if they are not equivalent.
Sets the current point as the ending point on the arc.
